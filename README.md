## 產品描述 / About

**Simple Gemini** is a lightweight Chrome extension for quickly summarizing selected text. 

This tool is designed to help users efficiently grasp the core points of any text.

### Key Features / 主要功能
- **Summarize selected text on webpages**: Highlight text and summarize it instantly.
- **Paste or type text**: Enter custom text directly into the popup to generate summaries.
- **Local Gemini Nano Support**: Use Chrome's built-in AI for private, local processing.
- **Gemini API Support**: Connect your own Gemini API key (BYOK) for cloud-based summaries.

## Extension Installation Tutorial / 擴充功能安裝教程 (解壓 ZIP, Load unpacked)

If you are installing Simple Gemini manually using the ZIP package, follow these steps:

### 1. Download and Extract / 下載與解壓
1. Go to the [Homepage](https://workingyuanyuan.github.io/simple-gemini-public/) or navigate to the `downloads/` folder in this repository.
2. Download the extension ZIP file (e.g., `simple-gemini-0.1.0.zip`).
3. Extract (unzip) the downloaded ZIP file to a folder on your computer.

### 2. Load Unpacked in Chrome / 載入已解壓的擴充功能
1. Open the Google Chrome browser.
2. In the address bar, type `chrome://extensions/` and press **Enter** to open the Extensions management page.
3. In the top-right corner of the Extensions page, switch the **Developer mode** (開發者模式) toggle to **On**.
4. In the top-left corner, click the **Load unpacked** (載入拆封的擴充功能 / 加載已解壓的擴充功能) button.
5. In the file picker dialog, select the **extracted folder** (the directory containing `manifest.json`).
6. The Simple Gemini extension is now successfully installed! You can pin the extension to your Chrome toolbar for easy access.
